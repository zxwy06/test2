# 星辰笔记个人博客

这是一个使用原生 HTML、CSS 和 JavaScript 实现的个人博客页面，包含顶部导航、文章展示、关于我、分类标签和页脚等模块。

## 项目文件

- `blog.html`：博客页面主体，包含页面结构、样式和交互脚本。

## 功能特点

- 响应式页面布局，适配桌面端和移动端浏览。
- 使用 CSS 变量统一管理颜色、阴影、圆角和字体。
- 顶部导航固定显示，方便快速跳转到不同页面区域。
- 文章卡片式展示，提升内容浏览体验。

## 运行方式

直接在浏览器中打开 `blog.html` 即可查看页面效果。

## 核心代码说明

- `:root` 中定义全局 CSS 变量，便于统一维护主题样式。
- `header` 和 `.nav-inner` 实现固定导航栏布局。
- `.logo`、`.nav-links` 等类负责页面头部的品牌和菜单展示。
- 页面主体通过 HTML 语义化标签组织内容，使结构更清晰，也便于后续维护。
